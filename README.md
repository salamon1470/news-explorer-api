# news-explorer-api

This is the API of the News Explorer app.

## Run the app

    npm run start
    
## Methods
    
    GET /users/me
    GET /articles
    POST /articles
    DELETE /articles/<articleId>
    POST /signup
    POST /signin
    
## Request example

`POST /signup`

## Response example
    
    {
      name:...,
      email:...,
    }
    
## Technologies used:

    Express-winston
    Helmet
    Mongoose

    And more..
